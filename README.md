# 🌶️ Advanced Personal Panel - 个人定制面板

高端个人导航面板 APP，简洁现代，支持 PC 和移动端自适应。

## ✨ 功能特性

- 🎨 **双主题支持** - 深色/浅色模式一键切换
- 📱 **响应式布局** - 完美适配手机、平板、电脑
- 🗂️ **分类管理** - 工作区/休闲区两侧分布，支持增删改
- 🔗 **网站管理** - 卡片式展示，支持增删改
- 🖱️ **拖拽添加** - 直接把桌面快捷方式拖入即可添加
- 💾 **本地存储** - 数据保存在浏览器 localStorage
- 🚀 **自动隐藏侧边栏** - 鼠标移入显示，移出隐藏

## 🚀 快速开始

### 方法一：直接打开
双击 `index.html` 文件即可在浏览器中运行

### 方法二：本地服务器（推荐）
```bash
# Python
cd apps/personal-panel
python -m http.server 8080

# Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 📖 使用说明

### 添加分类
1. 点击左侧（工作区）或右侧（休闲区）的「+ 分类」按钮
2. 输入分类名称和图标（emoji）
3. 选择区域，点击保存

### 添加网站
1. 点击分类卡片查看详情
2. 点击右上角「+ 添加网站」
3. 填写网站信息（名称、网址、图标、简介）
4. 点击保存

### 拖拽添加
直接把浏览器地址栏的链接或桌面快捷方式拖到中间内容区域即可

### 编辑/删除
- 鼠标悬停在分类卡片上，显示编辑/删除按钮
- 鼠标悬停在网站卡片上，显示编辑/删除按钮

### 主题切换
点击右上角的 🌙/☀️ 按钮切换深色/浅色模式
